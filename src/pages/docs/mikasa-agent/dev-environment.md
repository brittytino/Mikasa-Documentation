# Development Environment

Set up your development environment for Mikasa Agent development.

## IDE Setup

### VSCode Configuration
```json
{
  "mikasa.debug": true,
  "mikasa.trace": "verbose",
  "mikasa.format": {
    "enable": true
  }
}
```
