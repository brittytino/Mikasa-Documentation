# Running the Demo

Learn how to run and understand the Mikasa Agent demo application.

## Quick Start

```bash
# Clone the demo repository
git clone https://github.com/mikasa/demo
cd demo

# Install dependencies
npm install

# Run the demo
npm run demo
```

## Demo Features

1. **Basic Operations**
   - Agent initialization
   - Message handling
   - Extension loading

2. **Advanced Features**
   - Custom type handling
   - Event processing
   - Error recovery

3. **Monitoring**
   - Performance metrics
   - Resource usage
   - System health
